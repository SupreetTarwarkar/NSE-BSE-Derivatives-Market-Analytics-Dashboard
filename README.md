# NSE & BSE Derivatives Analytics Dashboard

## Project Overview

Power BI-based derivatives market analytics dashboard developed using NSE & BSE derivatives data. The dashboard helps analyze Open Interest trends, FII activity, participant positioning, Long/Short build-ups, and stock/index derivatives behavior to generate actionable trading insights.

## Objective

- Analyze Open Interest trends
- Track FII activity
- Monitor participant positioning
- Identify Long/Short build-ups
- Generate actionable trading insights using derivatives market data

## Tech Stack

- Power BI
- Power Query
- DAX
- Data Modeling
- Excel

## Data Source

- NSE Bhavcopy
- BSE Derivatives Data
- FII Activity Data
- Participant-wise OI Data
- Cash Mrket Deliery Data

## Dashboard Features

### Stock Delivery Analysis
- Delivery trends
- Trade activity monitoring

### F&O FII Derivatives Statistics
- FII Futures Positioning
- FII Options Positioning

### Long Short Ratio Analysis
- Long/Short positioning trends

### Index OI Statistics
- Strike-wise OI analysis
- Change in OI tracking

### Stock OI Statistics
- Stock-wise OI monitoring

### Future Stock OI Statistics
- Long Buildup
- Short Buildup
- Long Unwinding
- Short Covering

## Trading Insights

The dashboard enables traders to analyze Open Interest trends, participant positioning, FII activity, and derivatives market sentiment to support informed trading decisions.
